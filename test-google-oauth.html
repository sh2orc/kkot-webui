Google OAuth 직접 테스트

아래 버튼을 클릭하여 Google OAuth를 직접 테스트해보세요:

Google로 로그인 테스트

예상되는 결과:

  • ✅ Google 로그인 화면이 표시됨 → OAuth Client 설정 OK
  • ❌ "Error 400: redirect_uri_mismatch" → Redirect URI 설정 문제
  • ❌ "Access blocked" → OAuth consent screen의 Test users 미등록
  • ❌ "This app is blocked" → OAuth consent screen 설정 미완료
